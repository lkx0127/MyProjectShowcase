CTAI: Waste Classification Website using AWS

This website is designed to help individuals who lack recycling knowledge or are too busy to sort waste properly. It allows users to upload images of waste, and the system automatically classifies the waste type using AWS services. When users upload an image, the website—hosted on AWS S3—triggers REST APIs via AWS API Gateway. The API Gateway sends the image data to AWS Lambda, which processes the image using AWS Rekognition to determine the waste type. AWS Lambda then stores the classified data in AWS RDS (MySQL), allowing users to retrieve the results later. This project aims to make waste sorting easier for individuals, promoting better recycling habits by simplifying the process.

MLDP: Student Grade Prediction Model

This machine learning model predicts student performance based on various profile factors such as age, parental education, supportiveness, extracurricular activities, and tutoring. Multiple classifiers were tested using default parameters, with Gradient Boosting achieving the best performance. After fine-tuning the model, it reached an accuracy of 0.73. The model analyzes trends in student data, helping to identify those who may need additional support. Results are visualized using Streamlit, providing teachers with insights to personalize learning interventions for students.

RPA: HR System Automation

This project automates repetitive HR tasks, reducing workload using robotic process automation (RPA) with UiPath. The system scans incoming job applications from email, filters out spam, and processes resumes by checking for key fields like name, contact information, and qualifications. If a resume is incomplete, the system uses OpenAI GPT to identify missing details and sends an email requesting corrections. Suitable candidates are evaluated, and if approved, the system automatically generates offer letters and inputs the employee’s details into the HR onboarding system. This automation helps HR departments save significant time on resume processing, candidate evaluation, and onboarding.
